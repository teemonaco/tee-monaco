Key Features:

  • Discover: Curated research articles and newsletters.
  • My Stream: Personalized content feed based on user activity.
  • Search: Advanced search with detailed filters.
  • Account Settings: Customized news and updates.

Discover

The discover section features curated content, including trending research articles and newsletters. Users can explore detailed insights on topics like hepatocyte growth factors and save or share articles. This section is designed to enhance content discovery and engagement.

Discover

Search

The advanced search interface allows users to perform detailed searches using multiple filters and logical operators. This functionality ensures that users can find precise information quickly, tailored to their specific needs in various research areas.

Search Filter

Welcome to the SB Style Guide

This introductory screen of the SB Style Guide provides an overview of the code-ready component-based library developed using Atomic Design principles. It explains the atomic design methodology encompassing elements like ions, atoms, molecules, organisms, and complex organisms. Each class begins with the prefix "sb-" and uses the clamp/calc method for responsive padding, gaps, and font sizes.

Utility Classes

A collection of utility classes designed to help developers quickly adjust layout and styling without custom CSS. The page provides examples of responsive classes for rows, columns, padding, and margins. It emphasizes the use of flexible units and auto-resizing properties to create fluid, adaptable layouts.

Alerts

A guide to the alert components, which provide visual feedback to users. The page categorizes alerts by type (e.g., success, warning, error) and provides styling options for each. It also includes guidelines for usage, ensuring that alerts are used appropriately and effectively across the site.

Text & Color Classes

This section outlines the text styles and color classes, providing a range of sizes and styles that are easily adjustable. Developers can use these predefined classes to ensure consistent typography across the site. The page includes examples and usage guidelines for various text elements, ensuring they adhere to brand standards and accessibility requirements.

Themes

This page details the various themes available for the UI components, including guidelines on their application. Each theme is described in terms of purpose, usage, and visual style, providing a clear understanding of when and how to use them. The section also includes HTML and CSS examples for quick implementation.
