Welcome to My Portfolio

With over a decade of experience in user-centric design and development across e-commerce and B2B tech, I bring a unique blend of empathy and technical expertise, enriched by my background in nursing. This combination empowers me to create intuitive, impactful, and accessible digital experiences.

This portfolio is a personal project, entirely custom-coded by me to showcase my passion, dedication, and skills. Using HTML, CSS, JavaScript, and Bootstrap, I have meticulously crafted each element to demonstrate my commitment to delivering seamless, high-quality user experiences through meticulous design and innovative technology.

    • Spearheading UX/UI design and front-end development for the world's most visited papercrafting website.
    • Leveraging a unique blend of empathy and technical expertise to create intuitive and engaging user experiences, resulting in increased user satisfaction and conversion rates.

What Sets Me Apart

Empathetic User Learning

Harnessing the power of Looker Studio, I meticulously track user behavior and monitor customer support inquiries. This approach ensures that our products are tailored to meet the diverse needs of our users, enhancing their overall experience and satisfaction.

Technical Expertise

With proficiency in HTML, CSS, JavaScript, PHP, and Java (using Freemarker), I excel in creating seamless integrations between front-end and back-end technologies. My technical skills ensure robust and efficient web applications that deliver a superior user experience.

Design Expertise

Boasting over 10 years of experience in design tools like InVision, XD, and Figma, combined with 20 years of mastery in Adobe Creative Suite, I specialize in creating precise, scalable vector designs. My design expertise ensures visually appealing and highly functional user interfaces.

Analytical Skills

Crafting insightful dashboards by leveraging GA4 data through Looker Studio and Zapier, I drive informed design decisions. This analytical approach allows for real-time data visualization and actionable insights, ensuring that every design choice is backed by solid data and contributes to enhanced user engagement and business growth.

Welcome to the SB Style Guide

This introductory screen of the SB Style Guide provides an overview of the code-ready component-based library developed using Atomic Design principles. It explains the atomic design methodology encompassing elements like ions, atoms, molecules, organisms, and complex organisms. Each class begins with the prefix "sb-" and uses the clamp/calc method for responsive padding, gaps, and font sizes.

Utility Classes

The Utility Classes screen encourages developers to use flex and percent-based layouts over hard-coded pixel widths and heights. It showcases classes like sb-row and sb-col which can be nested and auto-resize padding and gaps. This section also provides examples of how these classes can be combined to create the perfect component, complete with HTML and CSS code snippets for easy copy and paste.

Text & Color Classes

This screen details various text and color utility classes that ensure all color combinations pass WCAG AA standards. It includes examples of resizable and fixed text sizes, text alignment, and non-wrapping text classes. Each class is accompanied by HTML and CSS code snippets, making it easy for developers to implement these styles consistently.

Themes

This screen showcases different themes that can be applied to components, including sb-th-gray, sb-th-white, etc. Each theme is designed to serve a specific purpose, such as providing a neutral backdrop or highlighting important elements. This screen provides visual examples and corresponding code snippets for each theme.

Buttons

The Buttons screen demonstrates the design and responsive scaling of various button styles using the clamp function. It covers button groups, different button sizes, and themed buttons like sb-btn-group, sb-btn-xl, and sb-btn-th-gray. Each example includes HTML and CSS code snippets for easy implementation.

Alerts

The Alerts screen provides different variations of alert components, such as success, error, warning, and informational alerts. Each alert type is designed to convey specific messages with distinct visual styles. The screen includes examples of nested and main component alerts, along with HTML and CSS code snippets.

Form Elements

The Form Elements screen details the input and select elements wrapped in sb-input-wrap classes for seamless icon integration. It includes examples of text fields, select dropdowns, email and phone inputs, and more. Each form element is designed for accessibility and consistency, with HTML and CSS code snippets provided for easy copy and paste.

Creative Kindness

The Creative Kindness screen presents a unique theme for the style guide, featuring custom color themes like ck-th-sand and ck-th-blue. It also showcases various assets, section layouts, and Font Awesome icons tailored to the theme. The screen provides a comprehensive view of how to implement this theme across different components, complete with HTML and CSS code snippets.
